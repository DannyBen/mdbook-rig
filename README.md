MDBook Rig
==================================================

Scaffolding for a new [mdbook][1] book using [rigit][2].

Usage
--------------------------------------------------

  $ rig install git@github.com:DannyBen/mdbook-rig.git
  $ mkdir book && cd book
  $ rig build mdbook


[1]: https://github.com/rust-lang/mdBook
[2]: https://dannyben.github.io/rigit/
